# Quickstart for Flask API Server

### 가상환경 진입 명령어
git bash :
```
source .env/Scripts/activate
```

cmd :
```
.env\Scripts\activate
```

### 가상환경 종료 명령어
cmd :
```
.env\Scripts\deactivate.bat
```
